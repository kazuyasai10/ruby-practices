終了条件
提出方法
Rubyで必須要件を満たすlsコマンドを書いてPull Requestとして提出する。
提出物にmacのlsコマンドの実行結果と併記して作ったlsコマンドの実行結果のスクリーンショットを貼る。スクリーンショットは-a -l -r -alrオプションそれぞれの実行結果を貼ること（-alrでも-lraでも-ralでも順番に関係なく実行できる必要があります）
rubocop-fjordのチェックが全てパスしたこともスクショで貼り付ける。
どうしても解決できない警告があれば、その理由を書くこと。
Pull Request 作成のための雛形リポジトリはこちら
https://github.com/fjordllc/ruby-practices
Pull Request はForkした自分自身のリポジトリに対して作成してください。

必須要件
gemを使わずにRubyの標準ライブラリのみで実装すること
メソッドを2つ以上使用すること
GithubでPull Requestとして提出すること
-a, -l, -rオプションを実装すること
-lオプションをつけた時の表示は揃えなくて良い
-alrでも-lraでも-ralでも順番に関係なく実行できること
rubocop-fjordをパスさせること
-lオプションをつけない場合は、横に最大3列を維持して表示する
# 横に最大3列を維持 = ターミナルの幅にかかわらずこんなふうに出力する
Gemfile           babel.config.js   log
Gemfile.lock      bin               package.json
Procfile          config            postcss.config.js
README.md         config.ru
